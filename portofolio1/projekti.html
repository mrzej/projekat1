Maturski rad

Isticanje važnosti životinjskih vrsta s kojima čovjek dijeli planet i koje su sve češće žrtve čovjekove sebičnosti, sadizma i nasilja. U svijetu je poznato 1,770.000 životinjskih vrsta, a pretpostavlja se da ih je gotovo stotinu puta više. Po podacima svjetskih organizacija, svaki dan nestanu stotine životinjskih vrsta. No, alarmantan podatak je da svakog dana zbog gubitka staništa, lova i iskorištavanja te onečišcenja atmosfere, tla i voda u svijetu izumre 50 – 100 vrsta biljaka i životinja. Napisati program koji za uneseni prirodan broj od 1 do 4 računa površinu,obim pravougaonika,kruga ili kvadrata.

      if n == 1:
    r = float(input('Unesite realan broj:'))
    from math import pi
    print(f'Povrsina kruga je: {((r**2)*pi):.2f}')
    print(f'Obim kruga je: {(2*r*pi):.2f}')
elif n == 2:
    a = int(input('Unesite prirodan broj a:'))
    b = int(input('Unesite prirodan broj b:'))
    print(f'Povrsina pravougaonika je: {a*b}')
    print(f'Obim pravougaonika je: {2 * a + 2 * b}')
elif n == 3:
    a = float(input('Unesite broj a:'))
    if a < 0:
        print('Broj je negativan! Pokusajte unijeti pozitivan broj.')
    else:
        print(f'Povrsina kvadrata je: {a**2}')
elif n == 4:
    a = int(input('Unesite cijeli broj a:'))
    b = int(input('Unesite cijeli broj b:'))
    if a > b:
        print('Broj a je veci od broja b!')
    elif b > a:
        print('Broj b je veci od broja a!')
    else:
        print('Brojevi su jednaki!')
else:
    print('Greska')
